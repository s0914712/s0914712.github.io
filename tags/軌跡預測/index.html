1. 概述 (What & Why)

NetCDF 是一套為「科學陣列型資料」設計的 自描述 (self-describing)、跨平台 (portable)、可擴充 (extensible) 的資料模型與檔案格式，同時也包含一組 API 函式庫。典型應用領域包含：氣象、海洋、氣候、遙測、模式模擬（如大氣模式、海流、浪場）等。
